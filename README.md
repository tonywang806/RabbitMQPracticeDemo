# RabbitMQPracticeDemo
## 概要

RabbitMQを基にして、メッセージ受信・発信処理のサンプルです。

## ライセンス

GNU GPL v2

## 機能

・メッセージ発送（個人別、部署別、全員）
・メッセージ受信（優先度も対応）

## 動作確認済み環境

・クライアント側

Windows 10 /　Windows 7

.NET Framework 4.5.2

・サーバー側

Windows Server 12

RabbitMQ
